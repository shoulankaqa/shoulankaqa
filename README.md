# 👋 Привет! Я Anka

Я **QA Manual** тестировщик, увлечённый качеством ПО и практическим тестированием.  
На этом GitHub репозитории я собираю **тестовую документацию, учебные тест-кейсы, баг-репорты и наработки QA**.

---

## Содержание репозитория

- **📑 Test Cases** – функциональные, негативные и позитивные тест-кейсы для разных проектов  
- **🐞 Bug Reports** – примеры найденных багов с описанием и скриншотами  
- **💻 Automation Scripts (по мере изучения)** – скрипты на JS для автоматизации тестов  
- **📚 Notes & References** – полезные материалы и заметки по QA  

---

## Цель репозитория

- Хранить все **практические материалы по тестированию** в одном месте  
- Показать потенциальным работодателям мои навыки и подход к QA  
- Систематизировать мой опыт тестирования и улучшать профессиональные навыки  

---

## Навыки

- **Тестирование ПО**: ручное тестирование, функциональное тестирование, негативное тестирование  
- **Документация**: составление тест-кейсов, баг-репортов, тест-планов  
- **Инструменты**: Jira, TestRail, Insomnia (для API), браузерные инструменты для QA  
- **Базовые навыки автоматизации**: JavaScript, Node.js, Playwright / Jest (в процессе изучения)
